## 🔧 RSSHub 可用路由

### 已验证可用

```bash
# 少数派 - 学习相关文章
http://localhost:1200/sspai/index

# V2EX - 教育话题（需筛选）
http://localhost:1200/v2ex/topics/hot

# Hacker News - 教育/AI相关
http://localhost:1200/hackernews/best
```

### 需要 Puppeteer（暂不可用）

```bash
# 知乎教育话题
http://localhost:1200/zhihu/topic/19550429/hot

# 知乎心理学话题
http://localhost:1200/zhihu/topic/19551647/hot

# 微信公众号
http://localhost:1200/wechat/mp/msgalbum/{biz}
```
