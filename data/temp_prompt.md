# 写作任务指引

## 核心命题
【人工智能“大事正在发生”：高等教育的人文主义重构】

## 目标受众
- 职场破局者、教育工作者、焦虑未来的家长
- 具备一定认知基础、对 AI/大模型浪潮有敏感度

## 核心视角（必须融入）
1. **“人机剥离对决”**：理工科死记硬背的知识正在被大模型无情包揽，“提问能力”、“人文感受”与“反思批判能力”成为教育系统内唯一不贬值的硬通货。
2. **“流水线的黄昏”**：探讨国内现阶段重技能培养、轻人文熏陶的“工厂流水线式教育”，在面对全知全能的 AI 时所暴露出的极端脆弱性。
3. **“重构时刻”**：这不是对机器的妥协，而是将人从工具属性释放后，重新回归到“人之所以为人”的最高形态。

## 写作规范
- **字数要求**：至少 1500 字，篇幅饱满。
- **文风调性**：深度、锐利、一针见血，带有《虎嗅》或《晚点》等新商业媒体的时代表达（抛弃生硬、假大空的学术用词和学生腔）。
- **结构编排**：使用 Markdown 语法进行层级清晰的章节划分，多用强有力的金句、Bullet Points（引用区块）。
- **输出格式**：不要包含多余的客套话，直接输出 Markdown 正文。
